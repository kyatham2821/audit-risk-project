# Audit Risk Intelligence System

A machine learning project that analyses financial 
statements from 500+ real public companies to 
automatically detect fraud risk and financial anomalies.

Built as part of my MSc in Data & Computational 
Science at University College Dublin.

---

##  Why This Project Exists

Auditors manually review hundreds of company 
financial statements looking for risk signals.
This project automates that process — flagging 
high risk companies instantly using data and ML.

---

##  What It Does

- Downloads real financial data from 500+ companies
- Computes industry standard fraud risk scores
- Detects unusual financial patterns automatically
- Groups companies by risk level using clustering
- Predicts audit restatement risk using ML models
- Visualises corporate relationship networks
- Presents everything in a live interactive dashboard
